# Polices de la charte

Déposer ici les fichiers de polices fournis par le club :

- `moonget.woff2` / `moonget.ttf` — titres (Moon get)
- `coolvetica.woff2` / `coolvetica.ttf` — corps de texte et menus (Coolvetica)

Les `@font-face` correspondants sont déclarés dans `app/assets/css/main.css`.
Tant que les fichiers ne sont pas là, le site retombe sur Quicksand / Jost
(Google Fonts), choisis pour leur proximité visuelle.

⚠️ Vérifier la licence web des deux polices avant la mise en production
(Moon get est souvent distribuée en "personal use only").
